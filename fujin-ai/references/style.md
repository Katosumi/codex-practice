# Fujin文体ルール

## 段落

- **1段落＝1〜2文** が基本。短い段落を積み重ねる
- 目安：40〜90字あたりが中心。長くても120字前後で切る
- 余韻のための短文単独段落をリズム調整に使う：「それが〜です。」「本題に入ります。」

## 見出し（noteスタイル）

- **`##` を中心に使う。`###` は補助のみ**
- 1記事で `##` を10個以上、`###` は2〜3個まで
- ラベル付き見出し（`【結論】`『【本題】』）は **1記事0〜1個** に制限
- まとめは `## まとめ` が定番
- CTAは `## 7日間で学べる無料のAI講座` が定番

## 太字・強調

- **本文中の太字は控えめに**。1セクション2〜3箇所まで
- キーワード・コンセプトの強調 → **太字のみ** で行う
- 読者の疑問・セリフの代弁 → 鉤括弧「」をそのまま使う（太字にしない）
- **NG**：`「**キーワード**」`（鉤括弧＋太字の併用は禁止）
- 太字マーカー `**` の前後に半角スペースを入れない：`〜は**強調語**です。` のように詰めて書く
- 数字は強く見せる（売上・期間・料金）。太字＋単位

## 箇条書き

- `-` を基本に、項目名を太字＋コロンで説明：`- **月契約の場合**: 月額1,900円`
- 手順は `**ステップ①〜**` のように太字で見出し化、直後に説明
- まとめセクションはハイフンなし・太字なしで1行1項目を列挙

## 絵文字

- 本文では基本「無し〜少なめ」
- CTAでは `👇` `🔽` `🚀` を限定的に
- 笑いは `w` 中心。多用しない

## 導入の型

```
どうも、Fujinです。

このnoteでは、海外のAIトレンドや最新情報など、次世代のテクノロジーをテーマにお届けしています。

最近、こんなふうに感じている人、多いんじゃないですか？

- （読者の悩み1）
- （読者の悩み2）
- （読者の悩み3）

今日は、（テーマ）について書いていきます。
```

短縮版：「このnoteでは〜」を省いて直接本題に入るパターンもOK。テンポ重視。

## 締めの型

```
## 7日間で学べる無料のAI講座

これからAIを活用していきたいけど、体型的にまとまっていて学べるものってないですよね。
そこでFujinが7日間で動画と音声で学べるかなり大ボリュームの講座を作りました。

下記のリンクから30秒で登録できますので、気になる方はどうぞ。

https://www.fujinmail.space/

ということで、今回はこの辺で終わりにしたいと思います。また次回の放送でお会いしましょう。じゃあね！
```

締めの3行は **改行せず1行に詰める**（投稿時の手作業を減らすため）。

## 記事構成の3型

### 1) 解説／How-to型

```
タイトル → 共感（読者の悩み） → 結論先出し → 背景／前提 → 手順1〜N → 落とし穴 → まとめ → CTA
```

### 2) トレンド／未来予測型

```
タイトル → 身近な導入 → 結論先出し（断言） → 前提（市場・技術の変化） → 理由1〜3 → 行動指針 → まとめ → CTA
```

### 3) 事例／ローンチ裏側型

```
タイトル → 数字の提示 → なぜ決断したか（ストーリー） → 戦略1〜3（裏側） → 再現性（抽象化） → まとめ → CTA
```

## 繰り返し表現の制御

### 削減すべき表現（連発禁止）

- **「ヤバい」系**：1記事3〜4回まで。連続使用NG
- **「全部」**：1セクション1回まで
- **「マジで」**：本当に強調したい箇所のみ

### 削除されやすい言い回し

- 自己評価系の前置き（「僕みたいに集客力のある発信者の場合」「僕レベルの発信者なら」）→ 削除
- 冗長修飾語（「先行で公開されている」→「公開されている」、「あえて言うなら」→ 不要）
- 平行構造の崩れ：「Aのアップデート と Bの登場で」→「Aの登場 と Bの登場で」（並列を揃える）

## ブランド表記

- **プラットフォーム名は英語表記**：`ブレイン` → `Brain`、`クロードコード` → `Claude Code`、`オープンクロー` → `OpenClaw`
- **長音「ー」を省略しない**：`ロンチ` → `ローンチ`
- **漢字とカタカナの選択**：`AIマンガ` → `AI漫画`（一般用語は漢字優先、商品名は正式表記を尊重）

## Xポストの型

### 王道：掛け算×ステップ

```
{ツールA} × {ツールB}（× {ツールC}）

{ベネフィット} 5ステップ

①{一手目}
②{二手目}
③{三手目}
④{四手目}
⑤{五手目}

{断定／総括（最強・革命・正しい）}
```

### ベネフィット語

`爆速` `爆上げ` `一撃` `大量生産` `超簡単` `完全自動化` `ロードマップ` `使い分け術`

### 締めの一言

`マジで最強になります` `できちゃうのマジ革命` `これがAI自動化の正しいステップ。`
`マジでGoogleエコシステム最強。` `AI時代にとっては最も重要。`

### CTA（リプ誘導／DM配布）

```
解説PDFは、いいねと【キーワード】とリプでDMします
```

キーワード例：`【自動化】` `【サムネ】` `【ノート】` `【使い分け】`

### 改行ルール（X特有）

- 1文ごと、または意味単位で改行
- 長い文は2行に分割
- `。` の一部を `！` に置き換えて熱量UP（控えめすぎない）

## チェックリスト（投稿前）

- [ ] 結論が冒頭に置かれているか
- [ ] 「自分はこう思う」の主観が入っているか
- [ ] 一般論だけで終わっていないか（経験・現場が織り込まれているか）
- [ ] 太字が多すぎないか（1セクション2〜3箇所まで）
- [ ] 「ヤバい」「マジで」「全部」が連発していないか
- [ ] 自己評価系の前置きが入っていないか
- [ ] ブランド名が正式表記か（Brain／Claude Code 等）
- [ ] 締めが1行に詰められているか
